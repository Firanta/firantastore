### 1. **Quality vs Size**
- Jika file terlalu besar, download resolusi lebih rendah

## 🎥 MEMBUAT VIDEO SENDIRI (Alternatif)

Jika tidak ada yang cocok, buat sendiri:

### **Tool Gratis:**
| Tool | Untuk | Difficulty |
|------|-------|-----------|
| **Blender** | 3D animation | ⭐⭐⭐⭐ |
| **DaVinci Resolve** | Video editing | ⭐⭐⭐ |
| **After Effects** | Motion graphics | ⭐⭐⭐⭐ |
| **Canva** | Simple graphics | ⭐ |
| **ffmpeg** | Conversion/processing | ⭐⭐⭐ |

### **Command line (ffmpeg):**
```bash
# Dari gambar sequence ke video
ffmpeg -framerate 30 -i frame_%04d.png -c:v libx264 output.mp4

# Create simple 10 second video dari color
ffmpeg -f lavfi -i color=c=purple:s=1920x1080:d=10 purple.mp4

# Create animated gradient
ffmpeg -f lavfi -i color=c=red:s=1920x1080:d=1,scale=1920:1080 \
  -c:v libx264 -crf 23 output.mp4
```
